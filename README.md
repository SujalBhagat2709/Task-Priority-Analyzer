# Task Priority Analyzer

## Overview

Task Priority Analyzer is a Python command-line application that helps users prioritize tasks based on importance, urgency, estimated effort, and due date. It calculates a priority score, assigns a priority level, and displays tasks in descending order of priority.

---

## Features

- Add Tasks
- Calculate Priority Score
- Importance Analysis
- Urgency Analysis
- Due Date Analysis
- Estimated Hours Analysis
- Priority Classification
- Sort Tasks by Priority
- Summary Dashboard
- Interactive Menu
